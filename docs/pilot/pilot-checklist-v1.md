# Tendo Pilot Checklist v1

## Before the pilot lesson

### Product setup

- [ ] Confirm production site opens: `https://tendo-ug.vercel.app`
- [ ] Check internet/device access.
- [ ] Open the four subject pages:
  - [ ] Mathematics
  - [ ] English
  - [ ] Integrated Science
  - [ ] Social Studies
- [ ] Open teacher worksheet generator.
- [ ] Generate one sample worksheet.
- [ ] Confirm printing/copying workflow works.

### Teacher setup

- [ ] Choose one class, preferably P7.
- [ ] Choose 1–2 teachers for first pilot use.
- [ ] Agree on pilot duration: one lesson, one week or two weeks.
- [ ] Choose the first topics to test.
- [ ] Explain known limitations honestly:
  - no login yet
  - device-local progress
  - videos mostly coming soon
  - teacher dashboard is early/local

### Suggested first topics

Good first demo topics:

- Maths: Venn diagrams, Fractions, Area, Money
- English: Letter writing, Environmental protection
- Science: Electricity and magnetism, Excretory system
- SST: Location of Africa, Climate of Africa

---

## During the pilot lesson

### Teacher-led flow

- [ ] Open one topic.
- [ ] Read the Big idea.
- [ ] Review Key words.
- [ ] Use Learn it in parts.
- [ ] Show the Visual brief / See it diagram.
- [ ] Discuss the PLE tip.
- [ ] Let learners answer Try this.
- [ ] Let learners take quiz/practice if devices allow.

### Observe learners

- [ ] Did learners understand the topic faster than with ordinary notes?
- [ ] Did learners ask fewer or better questions?
- [ ] Did the diagram help?
- [ ] Were any words too hard?
- [ ] Was the quiz level appropriate?
- [ ] Did the page feel too long, too short or about right?

### Observe teacher use

- [ ] Could the teacher navigate without help?
- [ ] Did the teacher use the diagram in explanation?
- [ ] Did the teacher trust the content?
- [ ] Did the teacher generate or want a worksheet?
- [ ] Did the teacher ask for a feature that is missing?

---

## After the pilot lesson

### Teacher feedback

Ask the teacher:

1. Which topic did you test?
2. What helped learners most?
3. What confused learners?
4. Was the language age-appropriate?
5. Were the diagrams useful?
6. Would you use the worksheet generator?
7. Which subject/topic should be improved first?
8. Would this save you preparation time?
9. Would you recommend this to another P7 teacher?
10. What would make this worth paying for?

### Learner feedback

Ask 3–5 learners:

1. Was the lesson easy to follow?
2. Which part helped most?
3. Which word or part was hard?
4. Did the diagram help?
5. Would you use this to revise at home?

### Founder notes

Record:

- [ ] School name
- [ ] Teacher name/role
- [ ] Class size
- [ ] Subject/topic tested
- [ ] Device setup
- [ ] What worked
- [ ] What failed/confused
- [ ] Teacher quote
- [ ] Learner quote
- [ ] Follow-up request

---

## Success signs

The pilot is promising if:

- teachers understand the product within 5 minutes
- teachers ask to use worksheets
- learners can explain the Big idea back
- diagrams help teachers explain faster
- teachers ask for more subjects/classes instead of only asking for videos
- teacher says it saves preparation or revision time

---

## Red flags

Investigate if:

- teachers do not trust the content
- learners cannot follow the language
- diagrams confuse more than help
- worksheet output is not classroom-ready
- teachers only see it as a toy, not a teaching tool
- navigation is unclear without founder help

---

## After-pilot decision

After the first pilot, decide one of three paths:

1. **Continue:** product is useful enough; improve based on feedback.
2. **Narrow:** one subject or teacher workflow is strongest; focus there.
3. **Fix first:** blockers are too serious; solve before broader rollout.
