# Teacher worksheet usefulness pass — 2026-07-01

## Verdict

The worksheet generator is now more teacher-ready after the Social Studies + English production push.

This pass focused on usefulness in a real P7 classroom: clearer subject pools, cleaner worksheet metadata, better print layout and a separate answer key.

---

## What changed

### Clearer teacher controls
- Added subject question-pool summary cards.
- Kept subject filters for:
  - Maths only
  - Science only
  - Social Studies only
  - English only
  - Mixed
- Added quick worksheet length buttons:
  - 10 questions
  - 15 questions
  - 20 questions
  - 30 questions
  - 40 questions
- Improved selected-topic summary to show how many questions are available from selected topics.

### Better worksheet details
Teachers can now set:
- school name
- class
- term/week
- time allowed
- marks per question
- custom instruction line

### Better answer-key handling
Before, answers could appear directly under each question.

Now:
- pupil questions print cleanly first
- answer key is separated into its own section
- print mode starts the answer key on a fresh page
- copied text also places the answer key after the questions

### Cleaner teacher page copy
The worksheet page now describes the pool as **question-bank topics**, avoiding confusion with total live topic pages across all subjects.
