*Date: 2026-06-30*
*Scope: audit the 10 topics previously considered the stronger study-ready set against `docs/spec/study-lesson-structure-v2.md` and the current P7 NCDC curriculum map.*

---

## Executive conclusion

These 10 topics are still the strongest group in the repo.

However, after applying the newer **study-lesson v2** standard, they should no longer all be treated as equally finished.

### New conclusion
- **2 topics are close to strong enough with minor edits only**
- **5 topics are solid but need a second-pass teaching rewrite**
- **3 topics need a deeper second pass because they are still too compressed for real self-study**

So the earlier label "study-ready" was useful relative to the weaker topics, but under the new standard it is **too generous** for some of them.

The founder's instinct was right again.

---

## Rating scale used

### A — strong, only minor polish needed
- clear teaching flow
- natural tone
- enough explanation for self-study
- internal structure already close to v2

### B — solid, but needs second-pass rewrite
- content is correct and useful
- lesson still reads a bit compressed or summary-like
- one more rewrite pass would improve real learning value a lot

### C — needs deeper second-pass rewrite
- correct content, but too compressed for independent study
- learner likely still needs more scaffolding, chunking, or examples

## Key pattern discovered

The main issue is not that the content is wrong.
The main issue is that the **heavier topics are still carrying too many sub-ideas in one page**.

That means there are now two different rewrite needs:

### Rewrite type 1 — polish rewrite
For topics already structurally sound but still slightly summary-like.
Examples:
- Perimeter
- Equations
- Decimals
- Area
- Volume
- Mean, median, mode and range

### Rewrite type 2 — depth/structure rewrite
For topics that may eventually need internal sub-lessons or much clearer chunking.
Examples:
- Fractions
- Proportion and percentages
- Money

## Product decision implication

The founder's earlier concern about one page being too little for a whole topic is now confirmed by the audit.

### Confirmed truth
Some topics can work as one well-written page.
Some cannot.

So the long-term study-ready product will probably need:
- better internal chunking immediately
- and eventually **sub-topic lesson navigation** for the heavier strands

---

## Bottom line

The stronger 10 topics are still the best material in the repo.
But under the newer study standard:
- only a few are close to fully ready
- most still need another pass
- the heaviest topics are still too compressed for true independent study

That gives us a very clear path forward.
